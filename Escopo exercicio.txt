Faça um sistema de marcação de ponto.

Parte 1

O sistema deve rodar enquanto o usuario não da o comando para encerrar.
Além disso, o sistema deve dar as seguintes opções para o usuário:

1- Iniciar jornada
2- Iniciar descanso
3- Retornar de descanso
4- Finalizar jornada

Para persistencia de dados, utilizar arquivo no formato .csv.

Ao finalizar o dia, o sistema deve gravar uma nova linha correspontente ao dia.
As informações que devem ser gravadas são:
- Nome da pessoa (Pode ser hard coded)
- Data/Hora inicio jornada
- Data/Hora inicio descanso
- Data/Hora retorno descanso
- Data/Hora fim jornada
- Tempo trabalhado no dia em horas

Dica: Criar classe base para leitura e escrita de dados(DataManagerBase) e criar uma especialização chamada FileDataManager, que irá implementar os métodos para lidar com arquivo .csv.


Parte 2
Crie uma base de dados local SQL para armazenar os dados da aplicação. Faça um script ou processo que passe os dados de .csv para a base de dados.

Parte 3
Atualize o sistema para ao invés de utilizar a classe FileDataManager, utilizar uma nova versão chamada DbDataManager, que herda da classe DataManagerBase.

Parte 4
Cada ação do usuário deve enviar um email para o mesmo, para servir de comprovante.

Parte 5
Crie um dashboard para exibir dados sobre a jornada
